JavaScript is a single threated , interpreted programming language , which can handle a single task at a time. JavaScript works asynchronous way , meaning it works to ensure non-blocking code execution. Asynchronous programming way is very useful for a faster execution of programs but it is difficult to develop. JavaScript uses different engines to run code in different browsers , such as- google chrome use v8 engine, Edge (Internet Explorer) use chakra , Mozilla Firefox use spider monkey, Safari use JavaScript core Web kit etc.

Event loop is a process to monitor bot the callback queue and the call stack .JavaScript works as a runtime model based on an event loop , which are working for executing codes , collecting and processing events , executing queued sub stack. Event loop generally works like a message processed completely before another . It passes message through queue , but if a message takes too much time to execute, it divided the message in many small parts so that the click or scroll are not get stacked.

The difference between sessionStorage and localStorage is the session storage data is cleared when the page session end but the localStorage data is not expire. The both storage are located in the browser. When a new tab open in the browser , a unique page session get created and assigned to that particular tab. Closing a tab/window ends the session and clears objects in sessionStorage but not in the localStorage . Both the storage save data in the same way which is (key value) way . We can access data from both storage .
